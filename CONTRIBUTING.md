# Contributing to Indian Exam AI Prompt Generator

Thank you for considering contributing to the Indian Exam AI Prompt Generator! We welcome contributions from everyone.

## Contribution Guidelines
1. **Fork the repository:**  Start by forking the repository to your GitHub account.
2. **Clone your fork:** Clone your forked repository to your local machine.
3. **Create a branch:** Always create a new branch for your feature or fix.
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make changes:** Make necessary changes in your local repository.
5. **Commit your changes:** Commit your changes with a clear message regarding what you’ve done.
   ```bash
   git commit -m "Brief description of your changes"
   ```
6. **Push to your fork:** Push your changes to your forked repository.
   ```bash
   git push origin feature/your-feature-name
   ```
7. **Create a Pull Request:** Navigate to the original repository and create a Pull Request from your branch.

## Code Style
- **Follow PEP 8:** Ensure your code adheres to [PEP 8](https://www.python.org/dev/peps/pep-0008/) style guide for Python.
- **Commenting:** Write clear and concise comments for your code.
- **Naming Conventions:** Use descriptive variable and function names.

## Ideas for Contribution
- Adding new features for generating exam prompts.
- Enhancing the existing AI prompt generation logic.
- Improving documentation and adding new examples.
- Implementing tests to ensure reliability.
- Updating dependencies and improving performance.

## Code of Conduct
Please adhere to the [Code of Conduct](CODE_OF_CONDUCT.md) of this project.

## Issues and Feedback
If you have any questions or face any issues, feel free to open an issue in the repository. We are here to help!
